text automata

Introduction

Text Automata allows users to quickly generate text from a template.

Toggle

Toggled text is text that a user may or may not want to exclude. The user can simply include (toggle on) or exclude (toggle off) the text in the UI when assembling the text. It is represented in the markdown using square brackets [ ].

Hi, [that time works well!][unfortunately, I won't be able to make it.]

Custom

Custom text allows the user to dynamically change a segment of text while generating the final text.

I would love to work at {name} as a {role}

Escape Character

The escape character '\' allows you to escape text that would otherwise be evaluated.

arr\[0] -> arr[0]
